基于hexo搭建个人博客-建站篇（win10）

发表于 2020-04-20 热度： Valine：
本文字数： 1.4k 阅读时长 ≈ 1 分钟

前言

我觉得，搭建属于自己的博客，很有必要。博客对考研复试、找工作面试等方面加成非常大。有人说博客已经过时了，但是我发现很多优质的资料都出自个人博客，这些博客都记录了博主遇到的问题和解决办法，还有自己的心得体会，这些都是宝贵的资源。正如某博主所说”对任何渴望进步的人来说，写博客/文章/回答对自己的成长帮助都是巨大的。频率和数量不必过分强求，有话要说就写写，没事就歇歇，随意点。因为坚持写作的那一刻起，你就已经开始受益。”搭建博客花了好几天，期间踩坑无数，遂记录下来以供后来者参考。

一、准备工作

1、下载安装node.js https://nodejs.org/en/

下载左边红色箭头标记的版本即可(下同)

键盘win+r，弹出运行框输入cmd，进入如下界面，分别输入node -v和npm -v，回车，显示版本号说明安装成功

2、下载安装git https://git-scm.com/downloads

安装后桌面右键会出现下图
git安装成功后需要配置邮箱、姓名和SSH密钥，网上有很多教程，在此不展开描述了，想要了解更多，详见廖雪峰git教程 https://www.liaoxuefeng.com/

3、注册github https://github.com/

(1) 注册后直接登录
(2) 新建一个仓库
(3) 创建仓库名：用户名小写.github.io，这就是博客创建成功后的访问地址，现在还不能用，创建好先放在这里。

二、安装hexo

1、cmd进入运行，输入下方代码利用淘宝镜像安装cnpm，等几分钟,完成后用cnpm -v检查是否安装完成

npm install -g cnpm --registy=http://registry.npm.taobao.org

2、输入下方代码正式安装hexo,完成后用hexo -v检查是否安装完成

cnpm install -g hexo-cli

3、自己挑一个盘(比如G盘)，命令行输入G:，再输入mkdir blog，打开G盘发现多了一个blog文件夹，回到命令行输入cd blog进入blog目录，输入代码hexo init，等几分钟

这里git必须是配置好邮箱和用户名，否则无法执行hexo init

下图说明本地hexo博客已经生成

4、输入hexo s，复制生成的地址转到浏览器粘贴打开，即可预览博客

三、部署

1、命令行输入cd blog进入blog文件夹，输入下方代码

cnpm install --save hexo-deployer-git

2、去硬盘打开blog文件夹，找到_config文件，用记事本或者其他编辑器打开，找到#Deployment按照下面格式输入，repo要填准备时自己新建仓库的地址

3、命令行输入hexo d将博客部署到GitHub仓库，部署时可能要输入github用户名和密码，密码输入时是看不见的，部署完成后浏览器输入自己的用户名小写.github.io并刷新，即可看见搭建的个人博客啦
